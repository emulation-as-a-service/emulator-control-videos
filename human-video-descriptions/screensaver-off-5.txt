Right-click on an empty inactive part of the desktop. A context menu will appear. Pick the item "Properties" by clicking on it. The "Display Properties" window will appear. In the top row of tabs, click on the one labeled "Screen Saver". A dropdown menu that allows to select the desired type of screen saver appears. Activate the menu and select the option "[None]". Next, click the button labeled "Power" top open the "Power Options Properties" window. Click the "Turn off monitor" dropdown menu and select "Never". Click the OK button to close the "Power Options Properties" window. and to return to the previous "Display Properties" window. Click the bottom "OK" button there to apply all changes made.
